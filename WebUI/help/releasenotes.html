Version 3.12.2

  • Haulier Enterprise ToughTouch Integration (HETTI): There is a new field on the driver screen ‘Passcode’ for use with a ToughTouch. This will be blank by default.
  • HETTI: There is a new flag on the vehicle screen to flag the vehicle as ‘ToughTouch installed’. This will be un-ticked by default.
  • HETTI: There are configuration settings for Mobile Worker Flow on each business type. These will be all set to ‘No’ by default.
  • HETTI: There are several new filters on the Traffic Sheet which are for Mobile Worker Flow enabled runs. Non ToughTouch users should ignore these and continue to use their business types/leg states filters as previously. Please be aware that if the instruction type ‘Non-MWF’ is unticked all of your instructions will disappear from the traffic sheet.
  • The ‘Booking In’ window on the collections/deliveries screen now defaults to the same booking time as the order.
  • Improvements and enhancement of the right hand search bar to speed up searches for orders.
  • Date pickers are now excluded from the tab order to remove impact on quick keyboard-centric data entry.
  • Bug Fix: The menu bar at the top of the screen now always display correctly in Internet Explorer.
  • Bug Fix: Filters no longer ‘hide’ behind reports in Internet Explorer.
  • Bug Fix: Post town now saves on drivers.
  • Changed default size of some pop-up windows to show all information.
  • Bug Fix: Client Invoicing Multi Batch screen invoice now always showing once approved.
  • Invoicing: Custom references can now be enabled or disabled individually per client via client settings > References tab.
  • Bug Fix: Change Booked Times from a run ‘Advanced’ mode fix.
  • Bug Fix: The ‘Enter Driver Start Times’ page close button now redirects to the View Start Times page.
  • Bug Fix: the ‘All Work for Sub-Contractor New’ report now retains data when a column sort is applied.
  • Added ‘Show more results’ button to the Auto Batch screen.
  • Increased the max number of cases which can be saved on an order from 9,999 to 99,999.
  • Columns on the Invoice Extra Preparation screen are now sortable.
  • Invoice Audit Trail filter now includes option to search by Invoice No.
  • Bug Fix: Invoice Date fields no longer stretch the full width of the screen.
  • Bug Fix: Running totals on the Multi Batch Invoice screen in the Ready to Invoice list are now correct when amount is above £1000. Note this bug did not affect the amount shown on the invoice.
  • Buttons at the top of the Add Order screen are now always mirrored at the bottom.
  • Change Request: There are two new fields on the driver screen: ‘Start Date’ and ‘Payroll No’. Neither are mandatory.
  • Change Request: The ‘Show location’ screen from the ‘My Resource’ section of the traffic sheet now has a time stamp.
  • Change Request: Schedule appointments can now be added for unavailable drivers.
